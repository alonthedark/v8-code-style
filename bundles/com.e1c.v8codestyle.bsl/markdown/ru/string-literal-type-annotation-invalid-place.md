# Теги размещены неправильно, внутри конструкции языка

## Неправильно

```bsl

Процедура НоваяПроцедура(ОбъектXDTO)
      
    Если Истина // @fqn    
    Тогда // @fqn
    
        Значение = "";  
    ИначеЕсли Ложь
      
    Тогда // @noN-nls-1
        
        Значение = "";
        
    Если
        Значение = "";
        
    КонецЕсли;    
     
КонецПроцедуры

```

## Правильно

```bsl

Процедура НоваяПроцедура(ОбъектXDTO)
      
    Если Истина Тогда 
    
        Значение = ""; // @fqn  
    ИначеЕсли Ложь
      
    Тогда 
        
        Значение = ""; // @noN-nls-1
        
    Если
        Значение = "";
        
    КонецЕсли;    
     
КонецПроцедуры

```
