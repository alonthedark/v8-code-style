# Присвоение переменной самой себе

Присвоение переменной самой себе не имеет смысла и обычно указывает на ошибку.

## Неправильно

Например, неправильно:

Если Ссылка.Метаданные().Имя = "ПоступлениеТоваровУслуг" Тогда

## Правильно

Например, правильно:

Если ТипЗнч(Ссылка) = Тип("ДокументСсылка.ПоступлениеТоваровУслуг") Тогда
